AspNetCore API Sample

Bem vindo à aplicação desenvolvida como exemplo/exercício de implementação de um sistema de cadastro de usuários. Este sistema utiliza:

  • ASP.NET Core e C# como Back-end.
  • Angular e TypeScript como Front-end
  • Bootstrap para estilização e layout.

O sistema suporta as seguintes funcionalidades:

  • Swagger: É utilizado para acessar a API de forma independente do Front-end, possibilitando visualizar e utilizar todas as rotas disponíveis
  • Usuários: É um componente que lista todos os usuários do sistema, além de realizar inclusões, edições e exclusões de usuários. Existem permissões especiais para acessar essas funcionalidades.
